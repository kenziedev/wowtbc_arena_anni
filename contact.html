문의 방법

사이트 관련 문의, 버그 신고, 데이터 수정 요청은 아래 방법을 통해 연락해 주세요.

GitHub

가장 빠른 문의 방법은 GitHub Issue를 등록하는 것입니다.

  • GitHub Issues — 버그 신고, 기능 제안, 데이터 관련 문의
  • GitHub Repository — 소스코드 및 프로젝트 정보

길드 / 캐릭터 추가 요청

리더보드에 길드 또는 캐릭터를 추가하려면 메인 페이지의 "+ 추가" 버튼을 이용하거나, 이슈 제목을 [추가]로 시작하는 GitHub Issue를 등록해 주세요.

데이터 삭제 요청

본인 캐릭터의 데이터 삭제를 원하시는 경우, GitHub Issue를 통해 요청해 주세요. 확인 후 처리해 드리겠습니다.
