About This Site

투기장 리더보드란?

WoW Classic TBC 20주년 기념서버(KR)의 투기장 PvP 순위를 추적하고 기록하는 커뮤니티 사이트입니다. 2v2, 3v3, 5v5 브라켓의 레이팅, 전적, 승률을 한눈에 확인할 수 있습니다.

주요 기능

  • 실시간 리더보드 — 약 6시간마다 Battle.net API에서 자동 갱신
  • 캐릭터 상세 정보 — 레이팅 변화 그래프, 장비, 특성 트리 확인
  • 검색 및 정렬 — 캐릭터, 길드, 직업 기반 검색
  • 길드/캐릭터 추가 — GitHub Issue를 통한 수집 대상 추가

데이터 출처

모든 게임 데이터는 Blizzard Battle.net API를 통해 공개적으로 수집됩니다. 길드 로스터에 포함된 캐릭터를 기준으로 PvP 통계를 추적합니다.
